# Horizon

A cinematic, keyboard-first presentation experience built with React.

![Build](https://img.shields.io/badge/build-passing-16a34a?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-0f172a?style=flat-square)
![Version](https://img.shields.io/badge/version-0.0.0-334155?style=flat-square)

## Features

- Smooth animated slide transitions
- Keyboard navigation (Left, Right, Space)
- Fullscreen presentation mode (F)
- Minimal, high-contrast visual system
- React + TypeScript + Vite foundation

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
